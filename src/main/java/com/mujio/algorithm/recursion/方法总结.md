## 利用数学归纳法推导递归写法

#### 数学归纳法：用于证明断言对所有自然数成立。

证明过程：

证明对于N=1成立
证明N>1时：假设对于N-1成立，那么对于N成立


````
对于: 1 + 2 + 3 +...+ n = n(n+1)/2

````
先证明第一个自然数成立，然后假设对于n-1成立，在用这个假设去推导证明对于n也成立

````
n = 1 时：
1 = 1 * 2 /2
如果 n-1时成立，
1 + 2 + 3 + ... + (n-1) = (n-1)n/2
则：
原式 = (n-1)n/2 + n 
    = (n(n-1) + 2n)/2
    = n(n+1)/2
    
````


递归写法：
1. 先一般，后边界
2. 缩小规模
3. 规模缩小1（广义上的1次，1层，1倍等）
